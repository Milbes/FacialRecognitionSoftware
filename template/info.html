This Module is a security based that will use and locate on individual's photo in the database.if the photo is available the screen will give an 'Access Granted' prompt signaling that the individual may proceed.On the other hand if an "Access Denied" prompt is shown then the individual will prohibited form further access.

- Next, user will select 'UPLOAD IMAGE' and upload an image of their choice.

- Once image is uploaded the user will see their uploaded image as preview

- The User will then select the "CHECK IMAGE" button and if given an "ACCESS GRANTED " prompt, they will be able to move on.
